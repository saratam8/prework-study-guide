HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Declarations are NOT case-sensitive. HTML elements ARE case-sensitive
  • Semantic HTML elements is where name of element describes its content and role it plays on a webpage. Best practice to use semantic HTML elements and easier to read by other developers.
  • Element refers to entire element from opening to closing tag
  • Tage refers to what appears inside the angle brackets

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules or declarations will apply.
  • A declaration comprises of a property and value.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: Adds a change in the working directory to the staging area
  • git commit -m "commit_msg": Commits staged changes to the local repo with a meaningful message about changes made
  • git pull origin main: Fetch and download content from main remote repo to the local repo
  • git push: Upload local repo changes to remote repo
